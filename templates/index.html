User secrets storage API

Valid routes:

  • /add/<secret> (addes a secret for authenticated user)
  • /view/ (displays all secrets for authenticated user)
  • /delete/<secret_id> (deletes a secret with specified id for authenticated user)
  • /update/<secret_id>/<new_secret> (updates a secret with specified id and specified new secret for authenticated user)

HTTP Digest Authentication Python Example:

url = 'http://api.pawel.pw/view/'
r = requests.get(url, auth=HTTPDigestAuth(<auth_user>,<password>))
